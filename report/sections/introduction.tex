\section{Introduction}

The introduction should provide background on the topic, state the problem or research question, explain the significance of the work, and provide an overview of the paper's structure. It should motivate the reader and set the context for the research.

IEEE guidelines:
\begin{itemize}
	\item Start with general background and narrow down to specific problem
	\item Clearly state the contribution of the paper
	\item Briefly outline the paper structure
	\item Define abbreviations on first use
\end{itemize}
