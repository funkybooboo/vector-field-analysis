# vector/tests

Catch2 tests for the `vector` library.

| File | Description |
|------|-------------|
| `test_vector.cpp` | Tests for `Vec2`: magnitude, unit vector, dot product, `almostParallel` |
| `test_streamline.cpp` | Tests for `Streamline`: initialisation and path growth |

## Running

```sh
mise run test:lib:vector
```
